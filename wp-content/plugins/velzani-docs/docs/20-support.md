# Suporte

Esta página contém informações sobre como obter ajuda e reportar problemas.

## Canais de Atendimento

Para questões técnicas ou dúvidas operacionais, utilize os seguintes canais:

- **Slack**: Canal #suporte-interno
- **E-mail**: suporte@velzani.com.br
- **Chamados**: Sistema de tickets interno

## Como Abrir um Chamado

1. Acesse o sistema de tickets
2. Selecione a categoria do problema
3. Descreva o problema com o máximo de detalhes
4. Anexe prints ou logs se necessário
5. Aguarde o retorno da equipe

## Horário de Atendimento

O suporte interno funciona de segunda a sexta, das 9h às 18h. Chamados urgentes fora do horário devem ser direcionados ao plantão via WhatsApp.

## Problemas Frequentes

### Acesso negado aos sistemas

Verifique se suas credenciais estão corretas. Caso o problema persista, solicite reset de senha pelo canal de suporte.

### Lentidão no sistema

Limpe o cache do navegador e tente novamente. Se persistir, reporte informando horário e ação realizada.
